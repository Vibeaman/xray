# XRay

X/Twitter account quality analyzer + PFP studio

## Features

### Account Analyzer
- **Quality Score** - Overall account rating (S/A/B/C/D tier)
- **Follower Analysis** - Real vs bot detection
- **Engagement Rate** - Avg interactions per post
- **Growth Trends** - Follower trajectory over time
- **AI Roast** - Fun roast of your profile
- **Shareable Card** - Download/share your stat card

### PFP Studio
- **Style Effects** - Apply artistic styles to your PFP
- **3D Render** - Turn flat PFP into 3D
- **Seasonal Themes** - Add seasonal decorations
- **Lore Generator** - AI-generated backstory for your PFP
- **PFP Merger** - Combine two PFPs (collab style)

## Tech Stack

- **Frontend:** React + Vite + TailwindCSS
- **Backend:** Node.js + Express
- **APIs:** Twitter/X API, OpenAI
- **Image Processing:** Sharp, Canvas

## Getting Started

```bash
# Install dependencies
npm install

# Start development
npm run dev
```
